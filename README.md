#  VirtualClassroomAI

> 从 Paper2Galgame 到沉浸式虚拟课堂：让学术论文/科学书籍变成可互动的 3D 教学场景

## 🌟 项目愿景
将枯燥的学术内容转化为**虚拟教室体验**：
- 📖 上传 PDF/电子书 → AI 智能分段解析
- 👩‍ 虚拟教师站在黑板前讲解重难点，支持实时板书/画图
- 🧑‍ 第一人称视角坐在课桌前，可举手提问、获得即时反馈
- 🌸 游戏化机制：虚拟同学互动、小红花奖励、知识图谱解锁
- 📚 扩展支持：不仅限于论文，覆盖教材/科普书/技术文档
